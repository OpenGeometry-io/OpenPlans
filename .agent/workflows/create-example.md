# Create a New Example

This workflow details how to create a new example HTML file.

1. Create a new HTML file under `examples/` (or `examples/elements/` appropriately).
2. Insert standard boilerplate structure.
3. Import the required shapes/elements from `../src/index.ts`.
4. Add basic Three.js lighting, controls, and `dat.gui` for configurability.
5. Provide a basic initialization script.
6. Check `npm run dev` to ensure the app loads it correctly at `localhost:5555/examples/your-new-example.html`.
