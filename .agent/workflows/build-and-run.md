# Build and Run Application

This workflow explains how to run the OpenPlans development server and build the application.

1. Ensure all dependencies are updated.
// turbo
2. Run the local development server:
```bash
npm install
npm run dev
```

3. To build the examples and distribution files:
// turbo
```bash
npm run build
npm run build-examples
```
